When Courses are Taken

This visualization shows how courses are distributed throughout the day.

You can choose which value to plot:

  • Course Count: The number of courses offered at a given time.
  • Enrollment: The number of students expected to be in class.
  • Capacity: The maximum number of students that could be in class.

You can also filter by department. This excludes cross-listed courses listed as belonging to another department.

The "normalize outputs" option will scale each department's graph so that they are all the same height. This may make cross-department timing comparisons easier.

Department Comparisons

This table summarizes several statistics about each department's course offerings. Click any column header to sort by that field.

Total Instruction Time refers to the total time spent by instructors teaching each week.

Cumulative Class Time refers to the total time students spend in class. Conceptually, this value can be thought of as being the hours of instruction weighted by enrollment.

Efficiency is the Cumulative Class Time / Instruction Time, which represents how many times the content is being duplicated by being listened to multiple people. This is similar to the average student count per section, but it isn't particularly beneficial to think about it that way.
