Course Curiosities at Princeton

There's a ton of information accessible through the office of the registrar page. This page is an attempt at synthesizing some of the Fall 2025 data.

All the data in this set was collected through a Python script using the princeton MobileApp API (more information).

When Courses are Taken

This graph shows information about courses throughout the course of the day.

The exact value plotted can be varied:

  • Course Count: The number of courses one could potentially take.
  • Enrollment: The number of students (supposed to be) in class.
  • Capacity: How many students could be in class.

Certain departments can be selected to see information about only classes offered by that department. Note that this refers to courses offered primarily by the department and doesn't include cross-listed courses.

The "normalize outputs" option will stretch subjects' graphs so that they are all the same height. This may help with comparing the time people take classes in a department.
